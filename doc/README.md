## 📖 文档说明

### 文档命名规范

所有文档文件名使用 **大写字母 + 下划线** 格式：
- ✅ `AI_SERVICE_GUIDE.md`
- ✅ `CHROME_AI_SETUP.md`
- ✅ `PROJECT_STRUCTURE.md`
- ❌ `ai-service-guide.md`
- ❌ `chromeAiSetup.md`

### 文档更新

- 每次功能更新后，及时更新相关文档
- 重大变更需要更新 `CHANGELOG.md`
- 新增功能需要更新 `TODO.md`
